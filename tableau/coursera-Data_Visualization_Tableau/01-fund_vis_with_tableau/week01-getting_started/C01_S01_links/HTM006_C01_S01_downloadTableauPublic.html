Você começará a explorar seus dados em minutos

Crie gráficos interativos, mapas fascinantes e painéis dinâmicos em questão de minutos. Salve sua visualização em seu perfil do Tableau Public e compartilhe-a onde quiser na Web. É extremamente fácil, qualquer pessoa pode usar e, ainda por cima, é grátis.

Tableau 2019.4 disponível para Windows e Mac

Requisitos do sistema

Destaques dos novos recursos do Tableau 2019.4

Melhorias em tabelas

Crie tabelas com até 50 colunas com a possibilidade de rolar horizontalmente. Para tabelas simples, classifique colunas inteiras por dimensões e campos discretos em distintos painéis.

Vá para onde uma planilha é usada

Estamos aumentando a facilidade para navegar por pastas de trabalho grandes e complexas com várias planilhas. Visualize com facilidade e vá diretamente para qualquer lugar de uma pasta de trabalho (painéis ou histórias) onde uma planilha é usada.

Mais recursos

Layouts automáticos para dispositivos móveis

Crie painéis ideais para dispositivos móveis em questão de segundos com os layouts automáticos para celular e tablet. Use os layouts como eles vêm, ou personalize-os para criar a melhor experiência móvel para seu painel.

Planilhas Google

No Tableau Public, queremos que seja possível acessar seus dados onde quer que eles estejam armazenados. É por isso que você pode se conectar do Tableau Public diretamente ao Planilhas Google. Além disso, se os seus dados no Planilhas Google forem atualizados, você poderá configurar sua visualização para ser atualizada automaticamente até uma vez por dia.

Explore todo o potencial da plataforma do Tableau

Da conexão à colaboração, o Tableau é a plataforma de análise completa mais avançada, segura e flexível para os seus dados. Oferece a profissionais dos dados os recursos ideais para seu nível de habilidades. Prepare, crie, explore ou visualize dados confiáveis com assinaturas da plataforma de análises governadas de autoatendimento do Tableau. Atualize para o Tableau Creator, uma oferta de assinatura que dá a você acesso ao Tableau Desktop, ao Tableau Prep e ao Tableau Server ou ao Tableau Online.
